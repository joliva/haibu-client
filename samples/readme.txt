Examples of starting an appliciation for different deployment types:

Type git:	example deploying from git server,		usage: hc start -c git_config.json
Type tar:	example deploying tar archive,			usage: hc start -c tar_config.json
Type zip:	example deploying zip archive, 			usage: hc start -c zip_config.json
Type local:	example deploying local application,		usage: cd local; hc start -c local_config.json
